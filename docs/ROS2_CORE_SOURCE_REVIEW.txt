    1: # D-CBF MPC Planner ROS2 Humble
    2: 
    3: A ROS2 Humble workspace port of the original ROS1 **MPC-D-CBF** planner.
    4: 
    5: - Original ROS1 project: <https://github.com/jianzhuozhuTHU/MPC-D-CBF>
    6: - Paper: <https://arxiv.org/abs/2209.08539>
    7: - Target platform: **Ubuntu 22.04 + ROS2 Humble + Gazebo Classic 11**
    8: 
    9: This repository preserves the paper/project pipeline: global path generation, local lidar map construction, obstacle ellipse fitting, Kalman prediction, and MPC with D-CBF obstacle constraints.
   10: 
   11: ## What is included
   12: 
   13: ```text
   14: src/
   15:   global_path_publisher/       Fixed straight-line /global_path publisher
   16:   linear_path_publisher/       Goal-driven /global_path publisher from /goal_pose
   17:   local_map/                   PointCloud2 -> grid map -> obstacle ellipses
   18:   obs_param/                   Kalman prediction for obstacle ellipses
   19:   local_planner/               CasADi/IPOPT MPC + D-CBF planner and controller
   20:   scene/                       Gazebo/RViz launch files and moving cylinder helpers
   21:   grid_map_cmake_helpers/      Vendored grid_map CMake helper package
   22:   grid_map_rviz_plugin/        Vendored RViz2 grid map display plugin
   23:   turtlebot3_velodyne_Gazebo/  Vendored TurtleBot3 + Velodyne Gazebo assets
   24: ```
   25: 
   26: The default full simulation uses `scene/worlds/world1.world`, which contains `cylinder_0` through `cylinder_4`. All five cylinders are configured to move through `scene/config/config.yaml`.
   27:

  140: ### Notes
  141: 
  142: - Use `scene sim.launch.py` for the complete project. `scene start_backup.launch.py` is only a Gazebo world launcher and does not start the planner/controller or moving-cylinder helper.
  143: - TurtleBot3's Gazebo diff-drive plugin already publishes `odom -> base_footprint`; keep `enable_odom_tf` at its default `false` unless you intentionally replace that TF path.
  144: - `local_planner/config/config.yaml` uses `safe_dist: 0.5` for TurtleBot3/Gazebo feasibility. If you increase it, verify that IPOPT still finds feasible solutions and that obstacle clearance remains acceptable.
  145: - `scene/config/config.yaml` sets `cylinder_num: 5`, so `cylinder_0` through `cylinder_4` move in the default `world1.world` scene.
  146: 
  147: ## Main topic contract
  148: 
  149: | Topic | Message | Producer | Consumer |
  150: | --- | --- | --- | --- |
  151: | `/goal_pose` | `geometry_msgs/msg/PoseStamped` | RViz / user | `linear_path_publisher` |
  152: | `/global_path` | `nav_msgs/msg/Path` | `global_path_publisher` or `linear_path_publisher` | `local_planner` |
  153: | `/velodyne_points` | `sensor_msgs/msg/PointCloud2` | Gazebo Velodyne | `local_map` |
  154: | `/for_obs_track` | `std_msgs/msg/Float32MultiArray` | `local_map` | `obs_param` |
  155: | `/obs_predict_pub` | `std_msgs/msg/Float32MultiArray` | `obs_param` | `local_planner` |
  156: | `/curr_state` | `std_msgs/msg/Float32MultiArray` | `controller` | `local_planner` |
  157: | `/local_plan` | `std_msgs/msg/Float32MultiArray` | `local_planner` | `controller` |
  158: | `/cmd_vel` | `geometry_msgs/msg/Twist` | `controller` | TurtleBot3 diff-drive plugin |
  159: | `/cmd_move` | `std_msgs/msg/Bool` | `local_planner` | moving cylinder helper |

    3: import rclpy.time
    4: import tf2_ros
    5: import math
    6: import numpy as np
    7: from geometry_msgs.msg import Twist
    8: from std_msgs.msg import Float32MultiArray
    9: 
   10: class Controller(Node):
   11:     def __init__(self):
   12:         super().__init__('controller')
   13:         self.declare_parameter('controller_horizon', 10)
   14:         self.declare_parameter('control_period', 0.02)
   15:         self.declare_parameter('parent_frame', 'world')
   16:         self.declare_parameter('child_frame', 'base_link')
   17: 
   18:         controller_horizon_value = self.get_parameter('controller_horizon').value
   19:         control_period_value = self.get_parameter('control_period').value
   20:         parent_frame_value = self.get_parameter('parent_frame').value
   21:         child_frame_value = self.get_parameter('child_frame').value
   22: 
   23:         self.N = max(1, int(10 if controller_horizon_value is None else controller_horizon_value))
   24:         control_period = float(0.02 if control_period_value is None else control_period_value)
   25:         self.parent_frame = str('world' if parent_frame_value is None else parent_frame_value)
   26:         self.child_frame = str('base_link' if child_frame_value is None else child_frame_value)
   27: 
   28:         self.rate = self.create_timer(control_period, self.control_loop)  # 50Hz by default
   29:         self.get_state = self.create_timer(control_period, self.get_current_state)  # 50Hz by default
   30: 
   31:         self.local_plan_sub = self.create_subscription(
   32:             Float32MultiArray,
   33:             '/local_plan',
   34:             self.local_planner_cb,
   35:             10)
   36:         self.vel_pub = self.create_publisher(Twist, '/cmd_vel', 10)
   37:         self.curr_state_pub = self.create_publisher(Float32MultiArray, '/curr_state', 10)
   38: 
   39:         self.tf_buffer = tf2_ros.Buffer()
   40:         self.listener = tf2_ros.TransformListener(self.tf_buffer, self)
   41: 
   42:         self.linear_speed = self.angular_speed = 0.0
   43:         self.local_plan = np.zeros([self.N, 2])
   44: 
   45:     def quart_to_rpy(self, x, y, z, w):
   46:         r = math.atan2(2*(w*x+y*z), 1-2*(x*x+y*y))
   47:         p = math.asin(2*(w*y-z*x))
   48:         y = math.atan2(2*(w*z+x*y), 1-2*(z*z+y*y))
   49:         return r, p, y
   50:     
   51:     def get_current_state(self):
   52:         try:
   53:             transform = self.tf_buffer.lookup_transform(self.parent_frame, self.child_frame,
   54:                                                         rclpy.time.Time())
   55:             _,_,yaw = self.quart_to_rpy(transform.transform.rotation.x,
   56:                                         transform.transform.rotation.y,
   57:                                         transform.transform.rotation.z,
   58:                                         transform.transform.rotation.w)
   59:             curr_state = Float32MultiArray()
   60:             curr_state.data = [transform.transform.translation.x,
   61:                                transform.transform.translation.y,
   62:                                yaw]
   63:             self.curr_state_pub.publish(curr_state)
   64:         except Exception as exc:
   65:             self.get_logger().debug(
   66:                 'Waiting for transform %s -> %s: %s' % (self.parent_frame, self.child_frame, exc)
   67:             )
   68:     
   69:     def pub_vel(self):
   70:         control_cmd = Twist()
   71:         control_cmd.linear.x = self.linear_speed
   72:         control_cmd.angular.z = self.angular_speed
   73:         # self.get_logger().info('Linear Speed: %.1f, Angular Speed: %.1f' % (self.linear_speed, self.angular_speed))
   74:         self.vel_pub.publish(control_cmd)
   75: 
   76:     def control_loop(self):
   77:         self.linear_speed = self.local_plan[0, 0]
   78:         self.angular_speed = self.local_plan[0, 1]
   79:         self.pub_vel()
   80: 
   81:     def local_planner_cb(self, msg):
   82:         available_pairs = min(self.N, len(msg.data) // 2)
   83:         if available_pairs == 0:
   84:             self.get_logger().debug('Received empty local_plan message')
   85:             return
   86: 
   87:         self.local_plan.fill(0.0)
   88:         for i in range(available_pairs):
   89:             self.local_plan[i, 0] = msg.data[0 + 2 * i]
   90:             self.local_plan[i, 1] = msg.data[1 + 2 * i]
   91: 
   92:         if available_pairs < self.N:
   93:             self.get_logger().debug(
   94:                 'Received %d local_plan pairs, expected %d' % (available_pairs, self.N)
   95:             )
   96: 
   97: def main(args = None):
   98:     rclpy.init(args=args)
   99:     controller = Controller()
  100:     rclpy.spin(controller)
  101:     controller.destroy_node()
  102:     rclpy.shutdown()
  103: 
  104: if __name__ == '__main__':
  105:     main()

  439:     Eigen::MatrixXf gradient_map_processing(Eigen::MatrixXf &map_data, vector<DBSCAN::Point> &dataset)
  440:     {
  441:         const float threshold = -1.25;
  442:         int col = map_data.cols(), row = map_data.rows();
  443:         Eigen::MatrixXf gradient_map(row, col);
  444:         gradient_map.setOnes();
  445:         gradient_map *= threshold;
  446:         DBSCAN::Point obs;
  447: 
  448:         for (int i = 1; i < row - 1; i++)
  449:         {
  450:             for (int j = 1; j < col - 1; j++)
  451:             {
  452:                 bool has_nan_value = false;
  453:                 for (int p = -1; p <= 1; p++)
  454:                 {
  455:                     for (int q = -1; q <= 1; q++)
  456:                     {
  457:                         if (isnan(map_data(i + p, j + q)))
  458:                         {
  459:                             gradient_map(i, j) = threshold;
  460:                             has_nan_value = true;
  461:                         }
  462:                     }
  463:                 }
  464:                 if (!has_nan_value)
  465:                 {
  466:                     float sobel_x = map_data(i + 1, j + 1) - map_data(i - 1, j + 1) + map_data(i + 1, j) - map_data(i - 1, j) +
  467:                                     map_data(i + 1, j - 1) - map_data(i - 1, j - 1);
  468:                     float sobel_y = map_data(i - 1, j + 1) - map_data(i - 1, j - 1) + map_data(i, j + 1) - map_data(i, j - 1) +
  469:                                     map_data(i + 1, j + 1) - map_data(i + 1, j - 1);
  470:                     gradient_map(i, j) = sqrt(sobel_x * sobel_x + sobel_y * sobel_y) + threshold;
  471:                     if (gradient_map(i, j) > obs_height + threshold)
  472:                     {
  473:                         obs.x = i;
  474:                         obs.y = j;
  475:                         dataset.push_back(obs);
  476:                     }
  477:                     else
  478:                     {
  479:                         Eigen::MatrixXf::Index minRow, minCol;
  480:                         Eigen::MatrixXf block = map_data.block(i / block_size, j / block_size, block_size, block_size);
  481:                         float min = block.minCoeff(&minRow, &minCol);
  482:                         if (map_data(i, j) - min > step_height)
  483:                         {
  484:                             obs.x = i;
  485:                             obs.y = j;
  486:                             dataset.push_back(obs);
  487:                         }
  488:                     }
  489:                 }
  490:             }
  491:         }
  492: 
  493:         return gradient_map;
  494:     }

   52: bool get_min_area_ellipse(const std::vector<Point> &pts, Ellipse &ellipse)
   53: {
   54: 	Min_ellipse me(pts.begin(), pts.end(), true);
   55: 	if (me.is_degenerate())
   56: 		return false;
   57: 	// r*x^2 + s*y^2 + t*x*y + u*x + v*y + w = 0
   58: 	double r, s, t, u, v, w;
   59: 	me.ellipse().double_coefficients(r, s, t, u, v, w);
   60: 	double a = -std::sqrt(2 * (r * v * v + s * u * u - t * u * v + (t * t - 4 * r * s) * w) *
   61: 						  ((r + s) + std::sqrt((r - s) * (r - s) + t * t))) /
   62: 			   (t * t - 4 * r * s);
   63: 	double b = -std::sqrt(2 * (r * v * v + s * u * u - t * u * v + (t * t - 4 * r * s) * w) *
   64: 						  ((r + s) - std::sqrt((r - s) * (r - s) + t * t))) /
   65: 			   (t * t - 4 * r * s);
   66: 	double cx = (2 * s * u - t * v) / (t * t - 4 * r * s);
   67: 	double cy = (2 * r * v - t * u) / (t * t - 4 * r * s);
   68: 	double theta;
   69: 	if (t != 0)
   70: 		theta = std::atan(1 / t * (s - r - std::sqrt((r - s) * (r - s) + t * t)));
   71: 	else if (r < s)
   72: 		theta = 0;
   73: 	else
   74: 		theta = M_PI;
   75: 	if (b > a)
   76: 	{
   77: 		swap(a, b);
   78: 		theta += M_PI / 2;
   79: 	}
   80: 	// if (t > 0.001)
   81: 	//   theta = std::atan(t / (s - r - std::sqrt((r - s) * (r - s) + t * t)));
   82: 	// else if (r - s < 0.001)
   83: 	//   theta = 0;
   84: 	// else
   85: 	//   theta = M_PI / 2;
   86: 
   87: 	ellipse = Ellipse{a, b, cx, cy, theta};
   88: 	return true;
   89: }

   91: Obstacle down_sample_points(const Obstacle &pts, int &x_min, int &y_min, int &x_max, int &y_max)
   92: {
   93: 	for (const auto &pt : pts)
   94: 	{
   95: 		if (pt(0) < x_min)
   96: 			x_min = pt(0);
   97: 		if (pt(1) < y_min)
   98: 			y_min = pt(1);
   99: 		if (pt(0) > x_max)
  100: 			x_max = pt(0);
  101: 		if (pt(1) > y_max)
  102: 			y_max = pt(1);
  103: 	}
  104: 
  105: 	int x_up = x_max - 0.1 * (x_max - x_min);
  106: 	x_up = x_up > x_min ? x_up : x_max;
  107: 	int x_down = x_min + 0.1 * (x_max - x_min);
  108: 	x_down = x_down < x_max ? x_down : x_min;
  109: 	int y_up = y_max - 0.1 * (y_max - y_min);
  110: 	y_up = y_up > y_min ? y_up : y_max;
  111: 	int y_down = y_min + 0.1 * (y_max - y_min);
  112: 	y_down = y_down < y_max ? y_down : y_min;
  113: 
  114: 	if (pts.size() > 100)
  115: 	{
  116: 		Obstacle ds_pts;
  117: 		for (const auto &pt : pts)
  118: 		{

  120: 				continue;
  121: 
  122: 			ds_pts.emplace_back(pt);

  178: void ab_variance_calculation(vector<Ellipse> &input_vector)
  179: {
  180: 	static vector<boost::circular_buffer<Eigen::Vector2d>> list;
  181: 	for (auto &input : input_vector)
  182: 	{
  183: 		int label = input.label;
  184: 		int current_size = list.size();
  185: 		if (current_size < label)
  186: 		{
  187: 			list.resize(label);
  188: 			for (size_t i = current_size; i < label; i++)
  189: 				list[i].set_capacity(10);
  190: 		}
  191: 		list[label - 1].push_back(Eigen::Vector2d(input.semimajor, input.semiminor));
  192: 		size_t list_size = list[label - 1].size();
  193: 		MatrixXd vac_mat(2, list_size);
  194: 		for (size_t i = 0; i < list_size; i++)
  195: 			vac_mat.col(i) = list[label - 1][i];
  196: 		MatrixXd zeroMeanMat = vac_mat.colwise() - vac_mat.rowwise().mean();
  197: 		MatrixXd covMat = (zeroMeanMat * zeroMeanMat.transpose()) / list_size;
  198: 		double variance = pow(covMat.trace(), 1.3);
  199: 		if (list_size < 3)
  200: 			variance += 0.0005 * (3 - list_size);
  201: 		input.variance = variance;
  202: 	}
  203: }

  175: void obs_kf::obs_predict()
  176: {
  177:     int num = param_list.size();
  178:     if (num == 0)
  179:         return;
  180: 
  181:     if (num == 1)
  182:     {
  183:         ka.m_x << param_list[0].x, param_list[0].y, param_list[0].a, param_list[0].b, param_list[0].theta, 0, 0, 0, 0;
  184:         ka.m_P = P0;
  185:     }
  186:     else
  187:     {
  188:         float _mea_cov_min = 0.005;
  189:         float _mea_cov_max = 0.05;
  190:         float _k_cov_min = 1;
  191:         float _k_cov_max = 3000;
  192:         float _mea_cov = min(max(param_list.back().mea_cov, _mea_cov_min), _mea_cov_max);
  193:         float _k = log10(_mea_cov / _mea_cov_min) / log10(_mea_cov_max / _mea_cov_min);
  194:         float _k_cov = pow(_k_cov_max, _k) * pow(_k_cov_min, 1 - _k);
  195: 
  196:         ka.m_R(0, 0) = _k_cov * R(0, 0);
  197:         ka.m_R(1, 1) = _k_cov * R(1, 1);
  198: 
  199:         VectorXd z(5);
  200:         z << param_list.back().x, param_list.back().y, param_list.back().a, param_list.back().b, param_list.back().theta;
  201:         param_list.erase(param_list.begin());
  202: 
  203:         ka.Predict_State();
  204:         ka.Predict_Cov();
  205:         ka.Mea_Resd(z);
  206:         ka.Cal_Gain();
  207:         ka.Update_State();
  208:         ka.Update_Cov();
  209: 
  210:         Kalman ka_tmp = ka;
  211: 
  212:         pred_list.clear();
  213:         for (int i = 0; i < N; i++)
  214:         {
  215:             ka_tmp.Predict_State();
  216:             ka_tmp.Predict_Cov();
  217:             obs_param pred;
  218:             float x_conf = 0.95 * sqrt(ka_tmp.m_P(0, 0));
  219:             float y_conf = 0.95 * sqrt(ka_tmp.m_P(1, 1));
  220:             float a_conf = 0.95 * sqrt(ka_tmp.m_P(2, 2));
  221:             float b_conf = 0.95 * sqrt(ka_tmp.m_P(3, 3));
  222:             float pos_conf = max(x_conf, y_conf);
  223:             float ab_conf = max(a_conf, b_conf);
  224: 
  225:             pred.x = ka_tmp.m_x(0);
  226:             pred.y = ka_tmp.m_x(1);
  227:             pred.a = ka_tmp.m_x(2) + pos_conf + ab_conf;
  228:             pred.b = ka_tmp.m_x(3) + pos_conf + ab_conf;
  229:             pred.theta = ka.m_x(4);
  230: 
  231:             pred_list.push_back(pred);
  232:         }
  233:     }

   20:         target_pos_ = Vector2d(this->get_parameter("end_x").get_value<double>(), this->get_parameter("end_y").get_value<double>());
   21: 
   22:         dist_ = (target_pos_ - start_pos_).norm();
   23:         diff_ = (target_pos_ - start_pos_) / dist_;
   24: 
   25:         RCLCPP_INFO(this->get_logger(), "start from (%f, %f) to (%f, %f)", start_pos_.x(), start_pos_.y(), target_pos_.x(), target_pos_.y());
   26:         RCLCPP_INFO(this->get_logger(), "step size = (%f, %f)", diff_.x(), diff_.y());
   27: 
   28:         step_ = 0.1;
   29:         timer_ = this->create_wall_timer(std::chrono::seconds(1), std::bind(&GlobalPathPublisher::timerCallback, this));
   30:     }
   31: 
   32: private:
   33:     void timerCallback()
   34:     {
   35:         nav_msgs::msg::Path global_path;
   36:         global_path.header.stamp = this->now();
   37:         global_path.header.frame_id = "world";
   38: 
   39:         geometry_msgs::msg::PoseStamped pose_stamped;
   40:         pose_stamped.header.stamp = this->now();
   41:         pose_stamped.header.frame_id = "world";
   42:         pose_stamped.pose.orientation.x = 0;
   43:         pose_stamped.pose.orientation.y = 0;
   44:         pose_stamped.pose.orientation.z = 0;
   45:         pose_stamped.pose.orientation.w = 1;
   46: 
   47:         for (double i = 0.0; i < dist_; i += step_)
   48:         {
   49:             Vector2d waypoint = start_pos_ + i * diff_;
   50:             pose_stamped.pose.position.x = waypoint.x();
   51:             pose_stamped.pose.position.y = waypoint.y();
